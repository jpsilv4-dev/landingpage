## 🚀 Como Publicar (3 Passos Simples!)

### Step 1: Personalizar (5 minutos) ⏱️

**Abra `index.html` em um editor e procure por:**

```
ANTES                          DEPOIS
────────────────────────────────────────
seu.email@example.com    →    seu.email@gmail.com
seu-usuario              →    seu-username-do-github
Unopar                   →    (deixe como está se não mudar)
IRED Internet            →    (deixe como está se não mudar)
```

**Salve após editar!**

---

### Step 2: Criar Repositório no GitHub (2 minutos) ⏱️

1. Acesse: **https://github.com/new**

2. Em **Repository name**, digite:
   ```
   SEU_USUARIO.github.io
   ```
   ⚠️ **Importante**: Sem espaços! Com `.github.io` no final!

3. Marque como **Public** ✅

4. ❌ NÃO marque **Add a README**

5. Clique em **Create repository**

---

### Step 3: Fazer Push (5 minutos) ⏱️

Abra PowerShell/Terminal e execute **na ordem**:

```powershell
# 1. Configure seu nome
git config --global user.name "Seu Nome Completo"

# 2. Configure seu email
git config --global user.email "seu.email@github.com"

# 3. Entre na pasta
cd d:\github\landingpage

# 4. Adicione o repositório remoto
# ⚠️ Substitua SEU_USUARIO pelo seu username
git remote add origin https://github.com/SEU_USUARIO/SEU_USUARIO.github.io.git

# 5. Configure a branch
git branch -M main

# 6. Envie para o GitHub
git push -u origin main
```

**Se pedir senha/token:**
- Vá em: https://github.com/settings/tokens/new
- Marque **repo** ✅
- Gere o token e cole no terminal

---

### Step 4: Ativar GitHub Pages 🌐

1. Acesse seu repositório
2. Clique em **Settings** (engrenagem)
3. Clique em **Pages** (lado esquerdo)
4. Verifique:
   - Branch: `main`
   - Pasta: `/ (root)`
5. **Salve**

⏳ **Aguarde 1-2 minutos...**

## 🔄 Fazer Alterações Depois

Quando quiser atualizar seu portfólio:

```powershell

# 2. Verifique as mudanças
git status

# 3. Adicione as mudanças
git add .

# 4. Faça commit com descrição
git commit -m "Sua descrição da mudança"

# 5. Envie para GitHub
git push origin main
```

As alterações aparecem online em 1-5 minutos!
