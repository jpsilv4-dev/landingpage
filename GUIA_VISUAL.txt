                        🔧 FAZER ALTERAÇÕES FUTURAS

═══════════════════════════════════════════════════════════════════════════════

Quando quiser atualizar seu portfólio no futuro:

1. Edite os arquivos localmente (index.html, style.css, etc)

2. Abra um PowerShell/Terminal e execute:

   cd d:\github\landingpage
   git status              # Ver o que mudou
   git add .               # Adicionar tudo
   git commit -m "Descrição da mudança aqui"
   git push origin main    # Enviar para GitHub

3. As alterações aparecem online em 1-5 minutos!

                        ❓ DÚVIDAS FREQUENTES

═══════════════════════════════════════════════════════════════════════════════

P: Quanto tempo leva para aparecer online?
R: Normalmente 1-2 minutos. Máximo 5 minutos. Limpe cache se demorar.

P: Posso mudar as cores?
R: Sim! Edite style.css e mude as cores em :root

P: Quero adicionar mais seções?
R: Edite index.html e copie/adapte as seções existentes

P: Como fazer um domínio personalizado (seu-nome.com)?
R: Vá em Settings → Pages → Custom domain e siga as instruções

P: Posso adicionar formulário de contato?
R: Sim! Use serviços como Formspree.io ou Netlify Forms

P: Como adicionar fotos/imagens?
R: Crie uma pasta 'assets' e referencie em style.css ou index.html
