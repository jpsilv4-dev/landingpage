# Portfólio - Landing Page Futurística

Uma landing page moderna e responsiva para portfólio profissional com design futurístico em gradiente amarelo-violeta.

## 🎨 Características

- ✨ Design futurístico com gradiente amarelo-violeta
- 📱 Totalmente responsivo (desktop, tablet, mobile)
- 🎯 Seções: Apresentação, Sobre, Habilidades, Jornada, Especializações e Contato
- 🚀 Animações suaves e efeitos interativos
- ⚡ Performance otimizada
- 🎭 Parallax effects e scroll animations
- 🔗 Links para redes sociais
- 💻 Código limpo e bem organizado

## 📁 Estrutura do Projeto

```
landingpage/
├── index.html      # Página principal
├── style.css       # Estilos e design
├── script.js       # Interatividades
└── README.md       # Este arquivo
```

## 🚀 Como Usar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/seu-nome.github.io.git
cd seu-nome.github.io
```

2. Abra o arquivo `index.html` em seu navegador ou use um servidor local:
```bash
# Com Python 3
python -m http.server 8000

# Com Python 2
python -m SimpleHTTPServer 8000

# Com Node.js (npm)
npx http-server
```

3. Acesse `http://localhost:8000` em seu navegador

## ✏️ Personalizando o Portfólio

### Informações Pessoais

Edite o arquivo `index.html` e atualize:

1. **Email** (procure por `seu.email@example.com`)
2. **GitHub** (procure por `seu-usuario`)
3. **LinkedIn** (procure por `seu-usuario`)
4. **Twitter** (procure por `seu-usuario`)
5. **Localização** (procure por `Arapongas, Paraná - Brasil`)

### Cores e Temas

As cores principais estão definidas no arquivo `style.css` nas variáveis CSS:

```css
:root {
    --primary: #FFD700;      /* Amarelo */
    --secondary: #9D4EDD;    /* Violeta */
    --dark: #0A0E27;         /* Fundo escuro */
    --darker: #050813;       /* Fundo mais escuro */
    --text: #E0E0E0;         /* Texto claro */
    /* ... outras variáveis */
}
```

### Adicionar Projetos

Para adicionar uma seção de projetos, edite o `index.html` e aumente a funcionalidade conforme necessário.

## 🌐 Publicar no GitHub Pages

### Opção 1: Repositório com nome de usuário (Recomendado)

1. Crie um repositório no GitHub com o nome: `seu-usuario.github.io`
   - Substitua `seu-usuario` pelo seu username do GitHub

2. Clone e copie os arquivos:
```bash
git clone https://github.com/seu-usuario/seu-usuario.github.io.git
cd seu-usuario.github.io
# Copie os arquivos index.html, style.css, script.js e README.md
```

3. Faça o commit e push:
```bash
git add .
git commit -m "Adicionar landing page do portfólio"
git push origin main
```

4. Acesse: `https://seu-usuario.github.io`

### Opção 2: Repositório com nome específico

1. Crie um repositório chamado `landingpage` ou outro nome

2. Vá para as **Settings** do repositório
3. Role para baixo até **GitHub Pages**
4. Em **Source**, selecione a branch `main` ou `master`
5. Clique em **Save**

6. Sua página estará disponível em: `https://seu-usuario.github.io/nome-do-repositorio`
